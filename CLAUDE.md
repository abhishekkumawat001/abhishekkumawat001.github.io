# Claude Code Instructions

This is Abhishek Kumawat's personal portfolio website built with Jekyll and the al-folio theme.

## Quick Reference

- **Site URL**: https://abhishekkumawat001.github.io
- **Theme**: al-folio (academic Jekyll theme)
- **Local dev**: `docker compose up` (runs at http://localhost:8080)

## Active Pages

- `_pages/about.md` - Homepage
- `_pages/implemented_papers.md` - Paper implementations
- `_pages/projects.md` - Project showcase
- `_pages/repositories.md` - Coding stats
- `_pages/cv.md` - Resume/CV

## Key Data Files

- `_data/cv.yml` - CV content
- `_data/repositories.yml` - GitHub repos list
- `_data/socials.yml` - Social media links
- `assets/json/resume.json` - Resume JSON data
