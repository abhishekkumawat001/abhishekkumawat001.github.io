## Overview

Implemented multi-class semantic segmentation using Convolutional Neural Networks with U-Net architecture featuring skip connections for precise pixel-level classification.

## Technical Details

- **Architecture**: U-Net with skip connections
- **Model**: Multi-class CNN for pixel-wise classification
- **Features**: Encoder-decoder architecture with feature concatenation
- **Framework**: PyTorch

## Key Features

- Skip connections for preserving spatial information
- Multi-class segmentation with softmax output
- Data augmentation for improved generalization
