## Overview

Conducted comprehensive analysis of transformer model compression through structured and unstructured pruning techniques on the T5 model, evaluating trade-offs between model size and performance.

## Technical Details

- **Model**: T5 (Text-to-Text Transfer Transformer)
- **Pruning Methods**: Structured pruning (attention heads, layers) and Unstructured pruning (weight magnitude)
- **Framework**: PyTorch, Hugging Face Transformers
- **Metrics**: Perplexity, inference speed, model size reduction

## Key Features

- Comparative analysis of pruning strategies
- Performance vs. compression trade-off evaluation
- Fine-tuning pruned models for recovery

## Technologies Used

`Python` `PyTorch` `Hugging Face` `T5` `Model Compression` `Pruning`
