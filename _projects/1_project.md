## Overview

Built a text classification pipeline using N-gram language models with configurable 1-5 grams for feature extraction and a Feed-Forward Neural Network (FNN) with 2 hidden layers for classification.

## Technical Details

- **Feature Extraction**: N-gram models (1-5 grams) for capturing local word patterns
- **Architecture**: Feed-Forward Neural Network with 2 hidden layers
- **Framework**: PyTorch

## Key Features

- Configurable N-gram range for flexible feature representation
- Multi-class classification with softmax output
- Efficient training pipeline with batch processing

## Technologies Used
