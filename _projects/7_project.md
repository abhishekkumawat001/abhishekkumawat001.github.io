## Overview

Developed a diabetes prediction system using Logistic Regression with dimensionality reduction techniques (PCA and LDA) for feature optimization and improved classification performance.

## Technical Details

- **Model**: Logistic Regression
- **Dimensionality Reduction**: PCA (Principal Component Analysis), LDA (Linear Discriminant Analysis)
- **Dataset**: Diabetes dataset with clinical features
- **Framework**: Scikit-learn

## Key Features

- Feature importance analysis
- Comparison of PCA vs LDA for dimensionality reduction
- Cross-validation for robust performance estimation

## Technologies Used

`Python` `Scikit-learn` `Pandas` `NumPy` `PCA` `LDA` `Logistic Regression`
