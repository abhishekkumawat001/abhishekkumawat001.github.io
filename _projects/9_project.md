## Overview

Built a credit card fraud detection system using Convolutional Neural Networks with techniques to handle class imbalance including SMOTE (Synthetic Minority Over-sampling Technique) and class reweighting.

## Technical Details

- **Model**: Convolutional Neural Network
- **Class Imbalance Handling**: SMOTE, Class reweighting
- **Evaluation**: Precision, Recall, F1-Score, AUC-ROC
- **Framework**: PyTorch, Imbalanced-learn

## Key Features

- Handling highly imbalanced dataset (fraud vs. non-fraud)
- Synthetic data generation with SMOTE
- Cost-sensitive learning with class weights

## Technologies Used

`Python` `PyTorch` `SMOTE` `CNN` `Fraud Detection` `Imbalanced Learning`
