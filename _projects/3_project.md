## Overview

Implemented character-level language models using multiple architectures (RNN, LSTM, FNN) with temperature-based sampling for controllable text generation.

## Technical Details

- **Architectures**: RNN, LSTM, and Feed-Forward Neural Networks
- **Sampling**: Temperature-based sampling for diversity control
- **Training**: Character-level tokenization and sequence modeling
- **Framework**: PyTorch

## Key Features

- Comparative analysis of different architectures
- Temperature parameter for controlling generation randomness
- Perplexity-based evaluation metrics
