## Overview

Developed a fashion recommendation system that uses visual similarity and multimodal AI to suggest similar fashion items based on image input, leveraging CLIP, CNN, and Google Gemini.

## Technical Details

- **Models**: CLIP (Contrastive Language-Image Pre-training), CNN for feature extraction
- **Multimodal AI**: Google Gemini for natural language understanding
- **Similarity**: Cosine similarity for visual matching
- **Framework**: PyTorch, OpenAI CLIP, Google Generative AI

## Key Features

- Image-based fashion item search
- Visual similarity scoring
- Natural language queries for fashion recommendations
- Cross-modal retrieval (text-to-image)
