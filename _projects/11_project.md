## Overview

Built a real-time cricket ball detection and tracking system using YOLOv8 for object detection and DeepSORT for multi-object tracking, with custom dataset created using Roboflow.

## Technical Details

- **Detection**: YOLOv8 (You Only Look Once v8)
- **Tracking**: DeepSORT (Deep Simple Online and Realtime Tracking)
- **Dataset**: Custom annotated cricket ball dataset via Roboflow
- **Framework**: Ultralytics, OpenCV

## Key Features

- Real-time ball detection in video frames
- Persistent tracking across frames
- Custom trained model for cricket ball detection
- Trajectory visualization

## Technologies Used

`Python` `YOLOv8` `DeepSORT` `Roboflow` `OpenCV` `Object Detection` `Tracking`
