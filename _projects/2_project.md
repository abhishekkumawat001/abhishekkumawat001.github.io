## Overview

Developed a sequence-to-sequence transliteration system using a GRU-based encoder-decoder architecture enhanced with Bahdanau Attention mechanism for improved accuracy.

## Technical Details

- **Encoder**: GRU-based recurrent neural network
- **Decoder**: GRU with attention-weighted context vectors
- **Attention**: Bahdanau (Additive) Attention mechanism
- **Framework**: PyTorch

## Key Features

- Attention visualization for interpretability
- Character-level sequence modeling
- Bidirectional encoder for better context capture

## Technologies Used

`Python` `PyTorch` `RNN` `GRU` `Attention Mechanism` `Seq2Seq`
