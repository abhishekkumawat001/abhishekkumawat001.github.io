## Overview

Applied unsupervised learning techniques to cluster MNIST handwritten digits, comparing performance of Gaussian Mixture Models (GMM), Bayesian GMM, and K-Means++ algorithms.

## Technical Details

- **Algorithms**: GMM, Bayesian GMM, K-Means++
- **Dataset**: MNIST (60,000 training images)
- **Evaluation**: Silhouette score, cluster purity, confusion matrix
- **Framework**: Scikit-learn

## Key Features

- Comparative analysis of clustering algorithms
- Dimensionality reduction for visualization (PCA, t-SNE)
- Automatic cluster number selection with Bayesian GMM

## Technologies Used

`Python` `Scikit-learn` `NumPy` `Clustering` `GMM` `K-Means`
